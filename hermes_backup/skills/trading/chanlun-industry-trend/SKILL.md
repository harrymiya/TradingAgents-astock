# chanlun-industry-trend

## 用途
当需要判断某个产业当前的景气度周期、跟踪产品涨价信号、发现放量爆发迹象、捕捉政策催化剂时，加载此 skill。

## 一、景气度周期四阶段模型

```
                    产能↑ 价格↑ 利润↑
       ┌─────────┐         ┌─────────┐
       │  导入期  │        │  放量期  │
       │ 概念先行 │───────►│ 业绩爆发 │
       │ 估值提升 │        │ 戴维斯双击│
       │ 少量受益 │        │ 全线受益  │
       └────┬────┘        └────┬────┘
            │                  │
            │    ┌─────────┐   │
            │    │  衰退期   │  │
            └───►│ 产能过剩  │◄─┘
                 │ 价格战    │
                 │ 行业出清  │
                 └──────────┘

            ┌──────────┐
            │  成熟期   │
            │ 增速放缓  │
            │ 龙头格局  │
            │ 龙头吃肉  │
            └──────────┘
```

### 各阶段特征与操作策略

| 阶段 | 营收增速 | 利润增速 | PE估值 | 股价表现 | 操作策略 |
|------|---------|---------|-------|---------|---------|
| **导入期** | <20% | 亏损或微利 | 极高（50-100x） | 概念炒作，波动大 | 只有缠论买点才参与，轻仓 |
| **放量期** | 50-200% | 100%+ | 30-50x（PEG<1） | 主升浪，持续上涨 | **重仓，缠论买点重仓干** |
| **成熟期** | 10-30% | 10-30% | 15-30x | 慢牛，稳健 | 正常仓位，长期持有 |
| **衰退期** | 负增长 | 亏损或下滑 | 10-20x | 持续下跌 | 不参与 |

### 如何判断当前处于哪个阶段？

**放量期信号（最值得操作）：**
- ✅ 行业龙头营收增速>50%且连续2个季度
- ✅ 产品涨价，供不应求
- ✅ 产能利用率>90%，扩产计划公布
- ✅ 多家公司业绩超预期
- ✅ 政策端明确支持

**导入期信号：**
- ✅ 技术突破但未商业化
- ✅ 券商研报密集推"0-1"
- ✅ 只有1-2家公司有相关收入
- ✅ 大量公司宣布"布局"

**衰退期信号：**
- ✅ 产品价格连续下跌
- ✅ 库存高企，去库缓慢
- ✅ 多家公司发布业绩预警

## 二、涨价信号捕捉（最重要的先行指标）

### 价格信号优先级

```
产品涨价 > 订单增加 > 产能满载 > 库存下降 > 客户扩产计划
```

**产品涨价是最直接、最可信的景气信号**。一个环节如果产品在涨价，说明供不应求，业绩暴增只是时间问题。

| 🏆 **800G光模块** | 🔥🔥 价格坚挺，需求翻倍 | 英伟达/谷歌大批量采购，2026年1.6T开始放量 | 中际旭创、新易盛、天孚通信 |

### 涨价信号的"三条腿"验证

```
看到涨价新闻 → web_search确认 → 查相关公司公告确认订单 → 三角验证通过
```

**一条涨价消息至少要找到两条以上独立来源确认。**

## 三、资金信号跟踪（验证景气度的辅助视角）

### 最可靠的资金信号

```
北向资金持续流入 > 机构调研频繁 > 龙虎榜机构买入 > 大宗交易溢价
```

### 北向资金关注清单（AI产业链）

需要定时用 `chanlun-market-data` 的 `get_northbound_flow()` 检查：
- 北向资金近5日净买入TOP10中是否有AI链公司
- 某只AI股北向持仓是否创近期新高
- 外资行研报是否上调目标价

### 产业资本信号

**最强烈的买入信号：**
- 公司回购（金额>1亿，且注销而非股权激励）
- 大股东/高管增持（真金白银买入）
- 产业资本举牌（同行看好）

**减仓信号：**
- 大股东/高管密集减持（即使股价很低）
- 产业资本减持（同行不看好）
- 定增折价发行（公司自己都卖不出好价）

## 四、政策催化跟踪（A股特色催化剂）

### 政策信号分级

| 级别 | 信号 | 影响时效 | 经典案例 |
|------|------|---------|---------|
| 🔥🔥🔥 **国家战略级** | 两会报告/五年规划/国常会 | 6-12个月 | "新质生产力"→AI全链上涨 |
| 🔥🔥🔥 **资金级** | 国家大基金注资/补贴 | 3-6个月 | 大基金一期→北方华创涨3年 |
| 🔥🔥 **部委级** | 工信部政策/发改委文件 | 1-3个月 | 算力枢纽政策→光模块暴涨 |
| 🔥 **地方级** | 地方政府规划/产业园 | 1-2周 | 地方算力补贴→短线脉冲 |
| ➖ **行业级** | 行业协会/标准制定 | 不确定 | 行业标准出台→龙头受益 |

### 当前需关注的政策方向（AI链）

1. **国家大基金三期**：投向重点包括HBM、AI芯片、先进封装、半导体设备
2. **新质生产力**：2024-2026年的顶层政策框架，AI+制造是核心
3. **数据要素×**：数据资产入表推进，利好数据基础设施
4. **算力基础设施**：东数西算二期，智算中心建设加速
5. **机器人产业政策**：人形机器人指导意见，各地产业规划密集出台

## 六、美股→A股映射关系（实战关键）

AI链的A股与美股高度联动，但不同类型标的受影响程度差异极大。需要区分"强映射"和"弱映射"：

### 映射强度分类

| 类型 | 逻辑 | 受影响程度 | 代表标的 |
|------|------|-----------|---------|
| **强映射（直接联动）** | 产品/技术直接对标美股，美国需求变化直接影响 | NVDA跌5% → 中际旭创次日跌3-7% | 中际旭创↔NVDA、海光↔AMD、澜起↔MU、长电↔TSM |
| **中映射（间接受益）** | 产业链配套，受整体景气度影响 | 美股大跌时跌2-5% | PCB（沪电）、散热（英维克）、材料（沪硅） |
| **弱映射（情绪影响）** | 国内逻辑独立，受美股情绪面短期冲击 | 美股大跌时跌0-2%或逆势 | 通信设备（中兴）、机器人（绿的谐波）、智驾（德赛西威） |

### 验证案例：2026年6月5日
- 美股：NVDA -4.40%、AMD -6.63%、AVGO -5.37%
- A股强映射：中际旭创 -7.81%、澜起科技 -7.81%、长电科技 -5.53%
- A股弱映射：**中兴通讯 +3.68%（逆势）**、绿的谐波 +20%（涨停）

### 实战应用
- 美股AI大跌的次日，**优先关注弱映射+自身有独立催化**的标的（如中兴有日线突破、绿的谐波有人形机器人独立逻辑）
- 强映射标的大跌时，**等跌幅消化后再看缠论买点**——恐慌性杀跌往往创造一买机会
- 可通过 `references/direct-http-api.md` 中的美股 K线API 获取 NVDA/AMD/AVGO 实时数据验证
