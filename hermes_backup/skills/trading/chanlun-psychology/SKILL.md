# chanlun-psychology

## 用途
当需要理解市场情绪、控制自身交易心态、识别潜在骗局和常见亏损模式时，加载此 skill。

## 市场心理研究（时代之）

### 情绪真相
- **恐惧不是在下跌时产生，而是在上涨初期** — "已经拉了一个涨停，现在追很危险"这才是恐惧
- **贪婪不是在追高时产生，而是在套牢时** — "再涨一点就卖"才是贪婪
- **损失厌恶**：套牢不肯割肉 → 账面亏损尚有希望 → 割肉等于承认失败

### 技术分析的最大敌人
- 技术分析基于概率，胜率永远不是 100%
- 人会因为一次失败就怀疑整个系统
- 没有哪个技术派对自己百分百相信 → 导致决策扭曲
- 机构用程序化交易解决情绪问题，但资金大幅回撤时连程序员都开始怀疑代码有bug

### 价值投资 vs 技术分析
- 技术分析需要极强的心理素质，频繁面对"我是不是错了"的自我怀疑
- 价值投资唯一对抗的是时间，但**寂寞往往最难耐**
- 真正价值派应该**越跌越高兴**（比亚迪例：跌30%时是加仓机会，后涨50%）

### 散户通病
1. 股价下跌还没到卖点就受不了卖出 → 蹭蹭往上涨 → 懊悔
2. 已发出卖出信号，想再等等 → 越套越深
3. 损失厌恶 → 套牢装死
4. 追涨杀跌 → 永远在高位买、低位割

## 123个亏钱案例教训

### 识人不明（31条）
- **信任成本远低于你想象** — 熟人是最容易坑你的人
- 熟人合伙 = 熟人坑
- 借信用卡给朋友 = 定时炸弹
- 老同学带你飞 = 可能是他在拿你当韭菜赚提成
- 叔叔有关系 = 拿你的钱套他的补贴

### 遇人不淑（55条）
- 所有"轻松赚钱"都是骗局
- 神秘项目、刷单、抖音出租、美女带你、高佣分成、微商代理、电商培训
- 要么货不对板，要么钱货两空，要么直接跑路

### 不自量力（37条）
- **不懂的东西不要碰**
- 别人在行业混这么多年都亏钱，凭啥你一个毛都不知道的人进去能赚钱？
- 股票发财梦、开奶茶店、P2P掏家底、买房暴富、虚拟币、跟风炒股
- **久赌必输** — 把时间线拉长，投机就是输的结局

### 一条金句
> "赚钱的操作千篇一律，亏钱的例子五花八门。这些案例不一定能帮你赚钱，但一定能帮你少亏不少钱。"
