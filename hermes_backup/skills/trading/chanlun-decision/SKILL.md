# chanlun-decision

## 用途
做最终投资决策时加载此 skill。它会加载其他 skill 的结论，综合判断并输出买入/持有/卖出信号 + 仓位建议。

**完整的选股→决策流程共7步，其中第0步（选股筛查）由 `chanlun-stock-screening` 完成**：
```
Step 0: 选股筛查 ← chanlun-stock-screening（批量扫描→评分排序→候选精炼）
Step 1-5: 五维交叉验证 ← 本 skill 的5个维度
Step 6: 综合决策 ← 本 skill 的决策矩阵
```

**在调用此 skill 前**，应先加载 `chanlun-stock-screening` 做初步筛选。扫描完成后从中精选2-4只标的，再逐只做本 skill 的五维交叉验证。

## 决策流程（6层递进）

### Step 1: 加载全部前置 skill
```
加载 chanlun-industry-chain   →  产业链位置 + 环节壁垒 + 龙头识别
加载 chanlun-industry-trend   →  景气周期 + 涨价信号 + 政策催化
加载 chanlun-value-research   →  基本面/选股结论
加载 chanlun-theory           →  技术面/缠论结论
加载 chanlun-market-data      →  最新市场实时数据
加载 chanlun-psychology       →  心理避坑
```

### Step 2: 数据层检查
从 chanlun-market-data 获取当前标的数据：
- K线走势
- 技术指标
- 新闻情绪
- 资金流向
- 北向资金
- 基本面
- 行业对比
- 龙虎榜/解禁（如果有异常）

从 chanlun-industry-trend 获取产业研判：
- 该股处于产业链哪个环节？
- 该环节当前景气周期阶段？
- 有无涨价/放量/政策催化信号？

### Step 3: 五维交叉验证

#### 维度①：产业链壁垒面（chanlun-industry-chain）
- 产业链位置：上游/中游/下游？微笑曲线哪端？
- 行业壁垒评分：技术/产能/客户/政策四维评分
- 卡脖子程度：国产化率多少？替代空间多大？
- 龙头地位：该环节A股龙头是谁？份额是否在提升？

#### 维度②：产业景气面（chanlun-industry-trend）
- 景气周期阶段：导入/放量/成熟/衰退？
- 涨价/放量信号：产品涨价？订单爆发？产能满载？
- 政策催化：有无国家级政策加持？
- 资金信号：北向资金/机构是否流入？

#### 维度③：价值面（chanlun-value-research）
- 行业属性：刚需消费/医疗/公用？还是银行地产科技？
- ROE > 12%？PE < 20？
- 国资背景？垄断性？
- 当前处于宏观周期的哪个阶段？
- 如果是科技股但处于放量爆发期，可以放宽PE标准（改看PEG）

#### 维度④：缠论技术面（chanlun-theory）
- 当前走势级别？日线/30F/5F？
- 中枢位置在哪？第几个中枢？
- 是否出现背驰？背驰类型（趋势背驰/盘整背驰）？
- 买卖点类型：一买/二买/三买？或者卖点？
- MACD状态：黄白线在0轴上还是下？是否回抽确认？
- 时间结构：1F≈3天周期，5F≈20-30天周期

#### 维度⑤：市场面（chanlun-market-data）
- 新闻面：有利好/利空？政策风向？
- 资金面：主力流入还是流出？北向资金态度？
- 情绪面：热股题材热度？散户情绪？

### Step 4: 综合决策矩阵

| 产业壁垒 | 产业景气 | 缠论信号 | 价值面 | 市场面 | 决策 | 仓位 |
|---------|---------|---------|--------|--------|------|------|
| 🔒高壁垒+卡脖子 | 🔥🔥放量爆发+涨价 | 一买/二买 | 通过 | 利好共振 | **重仓买入** | 8-10成 |
| 🔒高壁垒 | 🔥放量期 | 一买/二买 | 通过 | 中性偏多 | **买入** | 4-6成 |
| 🔒高壁垒 | 🔥放量期 | 三买 | 放宽通过 | 利好 | **买入** | 3-5成 |
| 中等壁垒 | 📈放量前期 | 一买/二买 | 待验证 | 中性 | **轻仓试仓** | 1-3成 |
| 中等壁垒 | 📈放量前期 | 盘整 | 通过 | 中性 | **观察** | 0-1成 |
| 低壁垒 | ➖导入期 | 任意买点 | 不通过 | 任意 | **谨慎/小仓** | 0-2成 |
| 任意 | 衰退期 | 任意 | 不通过 | 任意 | **不参与** | 0 |
| 任意 | 任意 | 一卖/二卖 | 任意 | 任意 | **卖出** | 清仓 |
| 任意 | 任意 | 趋势背驰 | 任意 | 任意 | **卖出** | 清仓 |
| 任意 | 任意 | 无明确信号 | 任意 | 利空 | **观望** | 维持 |

**特别规则（科技股放量期）**：
- 当产业趋势处于🔥放量爆发期且公司是产业链核心龙头时  
  → 可适当放宽PE<20的限制，改用 PEG<1 标准  
  → 但必须配合缠论买点，绝不在追高的位置买入

**各维度的优先级（决策时按此权重排列）**：
- 产业壁垒 > 产业景气 > 缠论技术 > 价值面 > 市场面
- 当高壁垒+高景气与价值面冲突时（如科技股PE高），放宽价值面标准
- 当缠论发出卖点时，其他维度无需再看——卖点至上

### 5. 心法约束（任何时候都要遵守）
- **不追高**：任何买入必须在买点上，没有买点不操作
- **级别匹配**：什么级别买的就在什么级别卖
- **宁卖早不卖晚**：卖点出现坚决走人
- **跌了只能考虑买，涨了只能考虑卖**
- **和情绪反着来**：散户恐惧时（涨初期）该买，散户贪婪时（高位）该卖
- **损失厌恶陷阱**：套牢不是不卖的理由，卖点出现必须卖
- **分散原则**：单只股票不超过总资金20%
- **不要有"等下一大级别"的想法**：只关心当下的分段操作
