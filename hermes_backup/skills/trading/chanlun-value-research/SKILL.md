# chanlun-value-research

## 用途
当需要进行股票基本面分析、估值判断、行业选择、长期投资标的筛选时，加载此 skill。

## 核心框架：盛京剑客投资体系

### 选股三要素
- **成长性** — 行业有长期增长空间
- **垄断性** — 市场地位难以被撼动
- **国资背景** — 有国家信用做后盾

### 投资公式
```
P = PE × E（股价 = 市盈率 × 每股收益）
```
- 公司质地 = 护城河 + 保护伞（国资/政策）
- 估值 = 安全边际（PE<20、PB尽可能低）
- 完美投资 = 质地优 + 安全边际高

### ROE 铁律
- ROE（净资产收益率）> 12% 为底线
- 净资产增长率决定长期股价走向

### 行业选择
**买什么**：14亿人每天用的刚需消费
- 猪肉、牛奶、榨菜、中药、酱油、醋、空调、免税
- 低端刚需消费、医疗医药、公共设施

**不买什么**：
- 银行/地产/保险（三者高度绑定）
- 科技股（需面对全球竞争）
- PE > 20

### 操作铁律
- 分散 20~30 只，长期持有
- 年初选10只优质股持有到年末
- 大盘下跌时不操作，谷底时操作
- 反散户操作：大资金愿意2800拿货，你为什么3700买？

### 宏观周期判断
- 衰退期→保守行业（医药、低端消费、公共设施）
- 轻民营、重国营
- 两会节奏：新提行业→建仓；去年已大涨今年还提→平仓
- 信息来源：新闻联播、部委网站
