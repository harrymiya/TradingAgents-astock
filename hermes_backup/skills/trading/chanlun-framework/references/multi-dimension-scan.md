# 全市场多维度扫描

## 位置

`tradingagents/dataflows/multi_dimension_scan.py`

## 架构

```
收盘后 → ensure_data() 检查DB完整性 → 
加载4360只股票日线 → 批量8维度评分(1秒) → 
按总分排序Top 50 → 保存scan_cache →
手动挑1-2只送入TradingAgents框架做完整分析(含游资+缠论辩论)
```

## 8 Analyst维度 → 量化筛选条件映射

| 维度 | 量化逻辑 | 分数 | 数据源 | 状态 |
|:----|:---------|:---:|:------|:---:|
| 市场 | 均线排列(10/20/60)、量比(5/20日)、5日涨跌幅 | 0-10 | DB K线 | ✅ |
| 情绪 | RSI(14)区间、MACD多空(DIF/DEA/柱) | 0-10 | DB K线 | ✅ |
| 事件 | 近10日涨停次数、放量大阳线(涨>5%+量比>1.5) | 0-10 | DB K线 | ✅ |
| 基本面 | PE/PB/营收增速 | 0-10 | ⏳待接腾讯/东财 | 暂返回5 |
| 政策/题材 | 同花顺热点题材归属 | 0-10 | ⏳待接get_hot_stocks() | 暂返回5 |
| 游资 | 成交量爆发指数(峰值/均值比)、连板天数(>3%连涨)、量价配合 | 0-10 | DB K线 | ✅ |
| 解禁/减持 | 限售解禁压力评估 | 0-10 | ⏳待接get_lockup_expiry() | 暂返回8 |
| 缠论 | 底分型识别、MACD底背驰/顶背驰简化版 | 0-10 | DB K线 | ✅ |

总分80分（当前基本面/政策/解禁暂返回中性，有效50分）。

## 评分函数详解

### 市场评分 (score_market_analyst)
- 均线排列：c > MA5 > MA10 +2, c > MA10 +1, MA10 > MA20 +1
- 量比：1.2~3倍温和放量 +1, >3倍放量过大 -1, <0.5缩量 -1
- 5日涨跌幅：+2%~+15% +1, <-5% -1

### 情绪评分 (score_sentiment_analyst)
- RSI(14)：40~70健康区间 +1, >80超买 -1, <25超卖 +1
- MACD：DIF>0 & MACD柱>0 +2(多头+红柱), DIF<0 & MACD柱<0 -1(双杀)

### 游资评分 (score_hot_money_analyst)
- 成交量爆发：近10日最大量 > 2倍20日平均 +2
- 缩量洗盘：近10日最小量 < 0.5倍20日平均 +1
- 放量大涨：近5日内某日涨>5%且量>2倍均值 +2
- 连板潜力：连续3日涨>3% +2, 连续2日 +1

### 缠论评分 (score_chanlun_analyst)
- 底分型：最后3根K线中间最低 +1
- 顶分型：最后3根K线中间最高 -1
- 底背驰：价格新低但MACD柱不新低 +2
- 顶背驰：价格新高但MACD柱不新高 -2

## 使用方式

```python
from tradingagents.dataflows.multi_dimension_scan import scan_market, show_candidates

# 全市场扫描，返回Top 50
results, date = scan_market(target_date="2026-06-05", top_n=50)

# 查看推荐候选
show_candidates(top_n=10, scan_date=date)
```
