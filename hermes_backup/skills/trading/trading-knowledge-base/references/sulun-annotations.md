## 素论 vs 缠论差异速查

| 方面 | 缠论 | 素论创新 |
|------|------|---------|
| 分析维度 | 空间几何结构 | 结构+时系+动能+筹码 四维 |
| 笔画标注 | 无统一代号 | 1B=笔, 0F=最小线段, 1F=1级线段... |
| 反转信号 | 背驰+三类买卖点 | 线段逆驰(快)+中枢逆驰(稳) |
| 时间分析 | 走势终完美(定性) | 时系六类关系(可量化) |
| 能量分析 | MACD背驰面积对比 | 三种动能:运行/空间/成区 |
| 中介状态 | 中枢/盘整 | 结构契合(临界态哲学解释) |
| 短线条件 | 无明确 | 5F向上/逆驰阶段+涨停板≥3只 |

## 逆驰信号强度排序

最强逆驰组合(日线):
1. 底分型 + MACD绿柱缩小(<0.85) + 最后收阳 + 放量>1.5倍 + 下跌段幅度>15% → 评分6-7/8
2. 底分型 + MACD红柱异常放大(逆驰动能) + 放量 → 评分5-6/8
3. 无标准底分型但有低位反弹 + 红柱出现 + 放量 → 评分3-4/8(弱信号)

## 素论短线三原则
1. 大盘处于5F向上走势类型中或逆驰阶段
2. 背驰段不宜短线建仓(防系统风险)
3. 涨停板<3只不开仓(ST除外)

## 时系六类识别(实战)
- 超前: 个股大级别超越大盘,小级别对抗大盘 → 最强
- 独立: 个股与大盘无关 → 独立行情
- 对立: 完全相反走势
- 主导: 个股走势带动大盘
- 跟随: 被动跟涨跟跌
- 滞后: 慢于大盘反应

> 当前nichi策略只实现了"结构+动能"维度, 时系+筹码维度需要额外数据源(大盘K线与个股同步 + 筹码分布数据)
