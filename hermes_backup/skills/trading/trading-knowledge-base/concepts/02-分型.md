# 缠论 分型

## 定义（来自缠师第62课原文）

**顶分型**：第二K线高点是相邻三K线高点中最高的，而低点也是相邻三K线低点中最高的。

**底分型**：第二K线低点是相邻三K线低点中最低的，而高点也是相邻三K线高点中最低的。

## 结构示意

```
顶分型:         底分型:
  /\              \  /
 /  \              \/
/    \              /\
       ↑顶        /  \
                  ↑底
```

## 关键规则

1. **顶分型的最高点**叫该分型的**顶**
2. **底分型的最低点**叫该分型的**底**
3. 顶和底是分型的简称（顶=顶分型的顶，底=底分型的底）
4. 分型必须在**包含处理之后**的K线上判断

## 分型强弱

### 顶分型强弱（决定是否成笔）
- **强顶分型**：第三K线跌破第一K线的最低点 → 大概率成笔
- **中顶分型**：第三K线跌破第一K线的收盘但不破最低 → 震荡
- **弱顶分型**：第三K线连第一K线收盘都不破 → 可能只是中继

### 底分型强弱
- **强底分型**：第三K线升破第一K线的最高点 → 大概率成笔
- **中底分型**：第三K线升破第一K线收盘但未破最高 → 震荡
- **弱底分型**：第三K线连第一K线收盘都不破 → 中继分型

## 分型与MACD辅助

- **底分型+MACD底背驰** → 第一类买点（大概率转折）
- **顶分型+MACD顶背驰** → 第一类卖点
- 分型可以出现在任何级别（1分钟/5分钟/日/周/月）

## 实战应用

分型是缠论的最小可识别单元，上连笔、中连线段、下连中枢：
```
K线包含处理 → 分型 → 笔 → 线段 → 中枢 → 走势类型
```
