# 缠论 笔

## 定义（来自缠师第62课原文）

**笔**：两个相邻的顶和底之间构成一笔。顶和底之间的其他波动都可以忽略不算。

### 基本要求
1. 一定是相邻的顶和底（隔了几个就不是了）
2. 顶和底之间至少有一根K线
3. 上升的一笔 = 底分型 + 上升K线（可多根）+ 顶分型
4. 下降的一笔 = 顶分型 + 下降K线（可多根）+ 底分型

### 新笔定义（放松版）
条件1：顶分型与底分型经过包含处理后，不允许共用K线（绝对不放松）
条件2：顶分型中最高K线和底分型的最低K线之间（不包括这两K线），不考虑包含关系，至少有3根（包括3根）以上K线

## 笔的方向

**向上笔**：底分型 → 一系列上升K线 → 顶分型
**向下笔**：顶分型 → 一系列下降K线 → 底分型

```
向上笔:       向下笔:
  顶            顶
  /\            /\
 /  \          /  \
/    \        /    \
     \       /
      \     /
       \   /
        底
```

## 关键规则

1. **顶和底之间至少有一根独立K线**（旧笔）或至少3根K线包含最高最低之间（新笔）
2. **必须相邻**：跳空形成的笔也需要有顶底分型
3. **边界K线可以共用**：一K线可以是前一笔的顶和后一笔的底（但顶和底不能是同一根）
4. 一笔至少需要5根K线（新笔：1根顶分型 + 3根中间K线 + 1根底分型）

## 笔的破坏

一笔被破坏意味着反向一笔已经形成：
- 向上笔被破坏 = 出现一个底分型且后续确认向下笔
- 向下笔被破坏 = 出现一个顶分型且后续确认向上笔

## 实战应用

笔是构建线段的最小单位。在实际操作中：
- **月线一笔** = 大级别趋势，对应数年的走势
- **周线一笔** = 中期趋势，对应数月的走势
- **日线一笔** = 短线趋势，对应数周的走势
- **30分钟一笔** = 超短线，对应几天
