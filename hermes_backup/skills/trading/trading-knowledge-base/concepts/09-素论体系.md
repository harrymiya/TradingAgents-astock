## 素论简介

素论自称"继波浪理论之后，最实用的操盘理论，技术分析界的奇门遁术"。核心声明：

> "永远记住我们出身缠门！永远敬仰伟大的禅师！……在先师的理论基础之上结合实战，本人总结了一些心得，实在是朴素无华，也未必符合缠师先宗，故称《素论》"

素论在缠论的基础上，将市场分析从**单一的结构分析扩展为四维体系**：

1. **结构（Structure）** — 缠论基础：包含处理、分型、笔、线段、中枢
2. **时序（Time-Series）** — 不同品种/板块间的时间先后关系（素论独创）
3. **动能（Kinetic Energy）** — 价格运动中的能量分类
4. **筹码（Chips/Distribution）** — 成本分布分析

> 核心哲学：**没有级别，不谈涨跌，不交易**

## 第0课：基础 — 结构构件

### 包含处理
与缠师原文一致：
- **方向定义**：第n-1到n根K线，高点抬高→向上，低点降低→向下
- **向上合并**：取高点中高点、低点中高点
- **向下合并**：取高点中低点、低点中低点

### 分型
- **底分型**：连续3根（包含处理后）K线，中间K线最高点最低、最低点最低
- **顶分型**：连续3根（包含处理后）K线，中间K线最高点最高、最低点最高

### 笔（最小几何线性单位）
- **向上笔**：底分型开始→顶分型结束
- **向下笔**：顶分型开始→底分型结束
- 顶底分型之间最少有**一根独立中间K线**
- 中间K线如果与分型元素有包含关系，**可以不处理**
- **笔是最小的几何递归级别，代号为 1B**

### 线段
- **最小级别（0F）线段**：至少由连续3笔组成，且前3笔必须有**重合区域**
- **1F线段**：至少由连续3根0F的段组成，且前3段必须有重合区域
- 类推到：5F、30F、RF、ZF、YF

### 级别递归
```
1B笔 → 0F段（3笔重合） → 1F段（3个0F段重合） → 5F段 → 30F段 → ...
```

## 第4课：结构契合（素论最关键的概念）

> "自然哲学和空间几何结构的结合，就是道与术的结合。……价格（指数）的内部结构揭示着自然哲学中的朴素道理，帮助我们看清市场（交易标的）的前生后世。"

### 太极思想在技术分析中的应用
- **太极生两仪，两仪生四象**：价格运行与万物相生相灭同理，阴阳制衡，相互转化
- **花开花谢的过程**：逐层逐级进行
- **没有级别，不谈涨跌，不交易**

结构契合是指走势运行到一种"临界状态"，即将发生方向转变。素论认为这不是随机的，而是结构本身的自然规律。

## 第5课：线段逆驰（素论独创概念）

### 线段逆驰（短线反转信号）
> "某级别的线段（X）运行到契合状态，进入了次级别的背驰阶段，突破向相反方向产生一股强大的反向动力，这种现象称之为线段逆驰！逆驰的出现标志着原来线段（X）的终结，新的反向同级别线段展开。"

#### 下降逆驰特征（下降→上涨）
- 次级别线段（a）上升角度达到45度及以上
- 连续K线向上冲击
- **成交量放大**
- **MACD红柱异常放大**，高度（或面积）很快超过（X）段内其他的上攻情形

#### 上升逆驰特征（上涨→下跌）
- 次级别（或次次级别）线段（a）下跌角度达到50度以下
- **突如其来的大阴线向下杀跌**（甚至跌停）
- **成交量过分放大**
- **MACD强势背驰**

### 中枢逆驰（趋势反转信号——更强势的反转信号）
> "某级别的线段（X）运行到契合状态，进入了次级别的背驰段阶段，在向相反方向逐步搭建一个同级别中枢，原线段运行趋势明显被削弱，这种现象称之为中枢逆驰！"

#### 中枢定义
- 某级别的走势类型，被至少三个连续次级别走势类型所重合的部分
- **上升中枢**：辗转反复、多头抵抗、积蓄上攻能量
- **下降中枢**：反复纠结、空头抵抗、积蓄下杀能量

#### 中枢分类（6种）
1. **顺势奔走型**
2. **逆势奔走型**
3. **标准型**
4. **平台型**
5. **三角型**
6. **逆势奔走型**（更强）

#### 中枢逆驰要素
1. 同级别反向线段要产生，必然要搭建同级别中枢
2. 符合反向线段产生的上一级别理由
3. X线段的背驰段c和逆驰段a的**角度越陡，冲击力度越大越好**，且非常清晰离开相应中枢

> **线段逆驰 vs 中枢逆驰的区别**：
> - 线段逆驰是**次级别直接反向**（快），适合短线交易
> - 中枢逆驰是更大级别的**逐步搭建中枢后再反向**（慢但更稳），适合中长线

## 第6课：时序（素论独创——将时间因素纳入分析）

素论指出："**在谈买卖点为时过早。等后面的时序和动能讲完后，再谈买卖点！**"

### 什么是时序
时系最初叫时序，但后来更正为"时系"——因为"反应的是一个事物对另一个事物的影响，或同一事物不同时期的影响关系"。

时序超越了品种、板块、大盘，甚至跨越不同的市场（包括外盘），是**另一个维度的神奇元素**。

### 核心原则
- **结构其实不是绝对的，是相对的**
- 大盘由个股组成，影响着个股；个股又牵连大盘
- 板块由个股组成，影响着个股；个股又牵连板块

### 时系六类关系（大盘与个股/板块与个股）
1. **独立** — 个股走势与大盘无关
2. **对立** — 个股走势与大盘相反
3. **超前** — 个股领先于大盘
4. **主导** — 个股主导大盘走势
5. **跟随** — 个股跟随大盘走势
6. **滞后** — 个股落后于大盘

### 上涨结构时系量化
- 中枢以外的区域可以细分为：
  - 逆驰段的起始位置
  - 中间位置
  - 结束位置

### 实战应用
- 大盘大级别向上+个股超前大盘 → 强势股
- "个股（板块）大级别超越大盘，小级别对抗大盘" → 最强势特征
- **多项通过对个股和大盘的时序对照，发现个股比大盘时序提前**
- **部分个股逆大盘时序提前**（独立行情特征）

## 第7课：动能分类（素论独创能量分析体系）

> "在实战中，我们发现即使结构和时序相同的个股走势也不一样，所以我们谈动能分类！"

### 三种动能
1. **运行动能** — 价格涨跌过程中所累积的能量。**背驰就是这种能量变化的比较过程**
2. **空间势能** — 价格所处的空间结构位置和运行角度而具有的潜在能量
3. **成区势能** — （OCR遗漏，但结合上下文指成交密集区的能量）

### 运行动能——实战指标
- **MACD红柱/绿柱面积对比**就是运行动能的直观体现
- 下跌段的绿柱面积 vs 前一段下跌的绿柱面积 → 判断背驰
- 上涨段的红柱面积 vs 前一段上涨的红柱面积 → 判断背驰

### 空间势能（素论独创）
- **观点一（逆驰势能）**：处于一个契合线段的背驰段，次级别反向冲击的势能可通过角度和位置量化
- **观点二（距离势能）**：向上契合段末端，距离最后一个中枢的位置越远，反向次级别段角度越大，则势能越大；向下同理

## 第8课：筹码分析

筹码分析基于成本分布：

### 两种观察方式
1. **筹码分布图** — 观察不同价位的持仓成本集中度
2. **量价变化** — 通过成交量判断筹码转移

### 关键指标
- 获利比例（%）
- 平均成本
- 90%成本集中度（上下沿价格区间）
- 70%成本集中度（更密集的区间）
- 不同时间段的成本占比（10日/30日/100日）

### 实战应用
成本集中度越低（数值越小）→ 筹码越集中 → 突破的爆发力越强

## 短线交易的素论原则

### 大盘安全条件
1. **5F向上走势类型中** 或 **相应级别的逆驰阶段**
2. **背驰段不宜短线建仓** — 防止系统性风险
3. 涨停板上少于3只股票（ST除外）→ **短线不宜开仓**

### 个股选择
1. **基本面检查**：了解行业/业务/业绩；没有超过2年及以上亏损，重大利空，ST或PT潜在风险
2. **题材概念**：符合未来发展方向、国家重点扶持、有发展前景者佳
3. **个股知名度**：历史上有强势表现、经常有强势表现的个股，股性更活，易于炒作
4. **读机构研报**更清晰了解（Ctrl+R查看）

## 素论核心口诀

> **结构看方向 → 时系看强度 → 动能看背驰 → 筹码看成色**

1. 先用**结构**确定大方向（上升/下降/盘整）
2. 再用**时系**判断个股与大盘的强弱关系（超前/跟随/独立）
3. 然后用**动能**确认背驰状态（运行动能是否衰竭）
4. 最后用**筹码**确认突破动力（成本是否集中）

## 素论 vs 缠论 对照

| 维度 | 缠论 | 素论 |
|------|------|------|
| 基础构件 | K线包含→分型→笔→线段→中枢 | 相同基础 + 代号系统(1B笔/0F段) |
| 分析维度 | 结构（空间几何） | **结构+时系+动能+筹码** 四维 |
| 时间维度 | 走势终完美（时间不确定） | **时系**：定量化跨品种时间关系 |
| 能量维度 | 背驰（MACD辅助） | **三种动能**：运行/空间/成区势能 |
| 反转信号 | 背驰+三类买卖点 | **线段逆驰+中枢逆驰**（更细致分类） |
| 中介状态 | 中枢/盘整 | **结构契合**（临界状态的哲学解释） |
| 买卖点 | 三类买卖点（精确位置） | 买卖点 = 结构+时系+动能三者共振 |
| 工具辅助 | MACD、均线 | 筹码分布图作为第4维验证 |
