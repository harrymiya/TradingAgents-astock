# 缠论 中枢

## 定义（来自缠师第17-18课）

**缠中说禅走势中枢**：某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分。

### 简化理解

```
            ┌─────┐
         ┌──┤ ZG  ├──┐
         │  │(高点)│  │
      ┌──┤  └─────┘  ├──┐
      │  │            │  │
      │  │  ┌─────┐  │  │
───┐  │  └──┤ ZD  ├──┘  │
   │  │     │(低点)│     │
   └──┤     └─────┘     │
      │                 │
      └─────────────────┘

    ZG = 三线段中最低高点的最低者
    ZD = 三线段中最高低点的最高者
    中枢区间 = [ZD, ZG]
```

### 关键参数
- **ZG**（中枢高）：构成中枢的三个线段中，各自高点的最小值
- **ZD**（中枢低）：构成中枢的三个线段中，各自低点的最大值
- 中枢区间 = [ZD, ZG]

## 中枢级别

| 本级别中枢 | 次级别走势 | 需要线段数 |
|-----------|-----------|-----------|
| 1分钟中枢 | 由1分钟线段构成 | 3段以上 |
| 5分钟中枢 | 由1分钟走势类型构成 | 3段以上 |
| 30分钟中枢 | 由5分钟走势类型构成 | 3段以上 |
| 日线中枢 | 由30分钟走势类型构成 | 3段以上 |
| 周线中枢 | 由日线走势类型构成 | 3段以上 |

## 中枢的形成与破坏

### 中枢形成
走势出现三段（或以上）次级别走势的区间重叠 → 中枢形成

### 中枢破坏（第三类买卖点）
- **第三类买点**：中枢形成后，次级别走势离开中枢，次级别走势回抽不进入中枢区间
- **第三类卖点**：中枢形成后，次级别走势离开中枢，次级别走势回抽不进入中枢区间

### 中枢的延伸
中枢形成后，后续走势继续在中枢区间内震荡 → 称为"中枢震荡"或"中枢延伸"

## 实战意义

1. **中枢是走势的"休息区"**：能量在集中，准备下一次方向选择
2. **中枢震荡**可以做短差（高抛低吸）
3. **中枢突破**（第三类买卖点）是趋势启动的确认信号
