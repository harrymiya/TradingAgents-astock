# 缠论 背驰

## 核心概念

**没有趋势，没有背驰。** 盘整中无所谓背驰。

## 定义

### 均线背驰（第15课）
缠中说禅趋势力度：前一段"吻"的结束与后一"吻"开始时，短线均线与长期均线相交形成的面积。

- 前后两个同向趋势中，后一个缠中说禅趋势力度比前一个弱 → 形成背驰

### MACD背驰辅助（最常用）
1. **黄白线（DIF）**：后一段趋势的DIF高点比前一段低 → 背驰
2. **红绿柱面积**：后一段柱子面积比前一段小 → 背驰
3. **柱子高度**：柱子不能继续伸长开始缩短时 → 严重关注

## 背驰分类

### 顶背驰（上涨趋势末端）
```
价格新高 + MACD红柱面积/高度 减小 + DIF不创新高
```

### 底背驰（下跌趋势末端）
```
价格新低 + MACD绿柱面积/高度 减小 + DIF不创新低
```

## 盘整背驰

在盘整（无趋势）中出现的背驰称为盘整背驰：
- 盘整背驰的力度比趋势背驰小
- 盘整背驰通常只产生震荡，不产生转折
- 盘整背驰+小级别转大级别 = 特殊转折

## 小级别背驰引发大级别转折（第44课）

小级别（如1分钟）的背驰，可能引发大级别（如5分钟/30分钟）的转折。

## 区间套（精准定位）

大级别背驰段出现后，去次级别找背驰点，再去次次级别找精确买点。

```
日线背驰段 → 30分钟背驰点 → 5分钟精确买点
```

## 背驰与买卖点的关系

- **趋势底背驰** → 第一类买点
- **趋势顶背驰** → 第一类卖点
- **盘整底背驰+第三类买点** → 最优买点组合
