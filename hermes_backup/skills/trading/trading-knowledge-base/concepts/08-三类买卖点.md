# 缠论 三类买卖点

## 第一类买点

**定义**：某级别下跌趋势中，最后一个次级别走势的底背驰点。

### 识别
1. 走势处于下跌趋势（至少两个中枢）
2. 最后一个中枢后的下跌段出现**底背驰**（MACD绿柱面积缩小、DIF不创新低）
3. 底分型+底背驰 = 第一类买点确认

### 实战
```
第一类买点 = 趋势底背驰
  是下跌趋势的终点
  是最安全的中长线买点
  需要次级别验证（区间套精准定位）
```

## 第二类买点

**定义**：第一类买点后，次级别回抽不创新低的位置。

### 识别
1. 第一类买点出现后，走势反弹
2. 随后出现次级别回调
3. 回调**不破**第一类买点的最低点
4. 回调止跌处 = 第二类买点

### 与第一类买点的关系
- 第二类买点比第一类买点**位置更高**但**确定性更强**
- 大级别的第二类买点 = 次级别的第一类买点

**缠中说禅买点定律**：大级别的第二类买点由次一级别相应走势的第一类买点构成。

## 第三类买点（最重要的买点）

**定义**：某级别中枢形成后，次级别走势离开中枢，然后次级别走势回抽**不进入中枢区间**。

### 识别
```
1. 先有中枢（三段次级别走势重叠）
2. 次级别走势向上突破中枢（离开段）
3. 次级别回抽的最低点 > 中枢的ZG（中枢上沿）
     ✓ 不进入 → 第三类买点成立
     ✗ 进入 → 中枢延伸
```

### 第三类买点的力度分类
- **强三买**：离开段力度很大（涨停、放量突破）+ 回抽段缩量且极小
- **中三买**：正常的离开+回抽
- **弱三买**：离开段力度弱，回抽靠近ZG

## 缠中说禅短差程序

**大级别买点介入的，在次级别第一类卖点出现时，可以先减仓，其后在次级别第一类买点出现时回补。**

## 实战应用（多级别联立）

以日线级别操作为例：
1. 日线出现第三类买点 → 买入
2. 等待30分钟出现第一类卖点 → 减仓
3. 30分钟出现第一类买点 → 回补
4. 日线出现第一类卖点 → 清仓

## 股票池中的三买判断

筛选出候选股票后，判断每只是否正处于三买结构中：
1. 是否有已形成的中枢？
2. 离开段是否力度足（放量/涨停）？
3. 当前是否处于回抽段？
4. 回抽是否已缩量止跌？
5. 回抽低点是否高于ZG？
