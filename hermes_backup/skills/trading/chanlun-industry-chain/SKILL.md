# chanlun-industry-chain

## 用途
当需要深入研究某个产业链的上下游结构、找到各环节的核心公司、判断哪个环节最有行业壁垒/护城河/定价权时，加载此 skill。

## 一、产业链研究的核心逻辑

### 为什么要研究产业链？

```
找到"卡脖子"环节 = 找到超额利润的来源
```

一个产业链中，利润最丰厚的环节通常满足以下特征：
1. **技术壁垒最高** — 别人做不了或做不好
2. **供需最紧张** — 想买买不到，产能不足
3. **替代难度最大** — 客户不敢轻易换供应商
4. **定价权最强** — 上游涨价能转嫁给下游
5. **集中度最高** — 行业份额集中在少数几家

### 六力定位模型

```
                    ┌──────────┐
                    │ 上游议价  │
                    │  能力    │
                    └────┬─────┘
                         │
   ┌──────────┐    ┌────┴─────┐    ┌──────────┐
   │ 潜在进入 │◄──►│            │◄──►│ 替代品   │
   │ 者威胁   │    │  本环节    │    │ 威胁     │
   └──────────┘    │  竞争     │    └──────────┘
                    └────┬─────┘
                         │
                    ┌────┴─────┐
                    │ 下游议价  │
                    │  能力    │
                    └──────────┘
```

**判断一个环节是否有壁垒，就看六力中它能挡住几个。**

### 产业链利润分布定律

```
微笑曲线：
       附加值
         ▲
         │   研发/设计     品牌/服务
         │      ╱╲            ╱╲
         │     ╱  ╲          ╱  ╲
         │    ╱    ╲        ╱    ╲
         │   ╱      ╲──────╱      ╲
         │  ╱        ╲    ╱        ╲
         │ ╱          ╲  ╱          ╲
         └─────────────────────────────► 产业链
           上游        中游       下游
         （材料/设备） （制造）  （品牌/应用）
```

- **上游核心材料和设备**：毛利率最高（50-80%），因为技术壁垒最深
- **中游制造**：毛利率最低（10-30%），因为竞争激烈、同质化
- **下游品牌和应用**：毛利率回升（30-60%），因为有品牌溢价和客户粘性

**关键洞察**：利润最高的不是"最热门的环节"，而是"最卡脖子的环节"。

## 二、产业链研究四步法

### Step 1：画产业链地图

以AI为例，逐层拆解：

```
                           AI 产业链全景
                               │
        ┌──────────────────────┼──────────────────────┐
        │                      │                      │
      上游                   中游                   下游
   （材料/设备/芯片）      （模型/平台）          （应用/终端）
        │                      │                      │
   ┌────┴────┐           ┌────┴────┐           ┌────┴────┐
   │ 硅片    │           │ 大模型   │           │ AI+办公  │
   │ 光刻胶   │           │ 算力平台  │           │ AI+医疗  │
   │ 特种气体 │           │ 数据服务  │           │ AI+教育  │
   │ 靶材    │           │         │           │ AI+安防  │
   │ 设备    │           │         │           │ 机器人   │
   │ GPU芯片 │           │         │           │ 智驾    │
   │ HBM存储 │           │         │           │ AI手机   │
   │ 光模块  │           │         │           │         │
   │ PCB    │           │         │           │         │
   │ 散热   │           │         │           │         │
   └────────┘           └────────┘           └────────┘
```

**具体每个环节再拆解：**

#### 上游：半导体设备产业链
```
光刻机（上海微电子） → 刻蚀（中微公司） → 薄膜沉积（拓荆科技）
  ↓                      ↓                     ↓
清洗（盛美上海/至纯科技） → 检测（中科飞测） → 离子注入（万业企业）
  ↓
涂胶显影（芯源微） → CMP（华海清科） → 分选/测试（长川科技/华峰测控）
```

#### 上游：芯片设计产业链
```
EDA工具（华大九天/概伦电子） → 逻辑芯片（海光/寒武纪/龙芯）
                               → 存储芯片（兆易/澜起/北京君正）
                               → 模拟芯片（圣邦/思瑞浦/纳芯微）
                               → MCU（中颖/兆易/国民技术）
```

#### 上游：先进封装产业链
```
前道设备（中微/北方华创） → 封测设备（长川/华峰） → 封装材料（康强/兴森）
                                  ↓
                       封测代工（长电/通富/华天）
```

### Step 2：评估每个环节的壁垒

使用"四维壁垒评分卡"，每个维度1-5分，满分20：

| 维度 | 评分标准 | 问自己 |
|------|---------|--------|
| 🔒 **技术壁垒** | 1-5分 | 这个环节需要多少年经验才能做？专利保护如何？ |
| 🏭 **产能壁垒** | 1-5分 | 扩产需要多长时间？投资规模多大？ |
| 🤝 **客户粘性** | 1-5分 | 客户换供应商的成本有多大？认证周期多长？ |
| 📜 **政策壁垒** | 1-5分 | 有没有许可证、资质、安全认证要求？ |

**评分示例：**

| 环节 | 技术壁垒 | 产能壁垒 | 客户粘性 | 政策壁垒 | 总分 | 判断 |
|------|---------|---------|---------|---------|------|------|
| 光刻机 | 5 | 5 | 5 | 4 | **19** | 🔒极高壁垒 |
| GPU芯片 | 5 | 4 | 5 | 3 | **17** | 🔒极高壁垒 |
| 光模块 | 3 | 2 | 3 | 1 | **9** | 中等壁垒 |
| 封测代工 | 2 | 3 | 3 | 1 | **9** | 中等壁垒 |
| 机器人执行器 | 4 | 3 | 4 | 2 | **13** | 较高壁垒 |
| AI服务器组装 | 2 | 1 | 2 | 1 | **6** | 低壁垒 |
| 液冷散热 | 3 | 2 | 3 | 1 | **9** | 中等壁垒 |

**只有总分≥12的环节才值得深入追踪。**

### Step 3：判断"卡脖子"程度

```
国产化率 < 30% → 严重卡脖子 → 国产替代空间最大 → 政策支持最强
国产化率 30-60% → 正在突破 → 已经跑出龙头 → 竞争加剧
国产化率 > 60% → 基本完成替代 → 剩下存量竞争 → 关注龙头份额提升
```

**AI产业链的国产化率现状（参考）：**

| 环节 | 国产化率 | 卡脖子程度 | 替代空间 |
|------|---------|-----------|---------|
| 光刻机（高端） | <5% | 🔴 极度卡脖子 | 巨大但难度极高 |
| EDA软件 | 15% | 🔴 严重卡脖子 | 巨大 |
| GPU/AI芯片 | 20% | 🔴 严重卡脖子 | 巨大 |
| 半导体设备（整体） | 25% | 🟡 显著卡脖子 | 大 |
| HBM/高带宽存储 | 15% | 🔴 严重卡脖子 | 巨大 |
| 光模块 | 60% | 🟢 已占优势 | 全球领先 |
| 先进封装 | 35% | 🟡 正在突破 | 大 |
| 半导体材料 | 20% | 🔴 严重卡脖子 | 巨大 |
| 晶圆代工（先进制程） | 20% | 🔴 严重卡脖子 | 巨大 |

### Step 4：找到"微笑曲线"的高端

**在AI产业链中，微笑曲线两端的位置决定了利润分配：**

```
毛利率 80% ┊
         ┊  光刻机/设备
         ┊    │
  60%    ┊    ├── GPU/IP/EDA
         ┊    │     │
  40%    ┊    │     ├── 材料/存储  品牌应用（海康/金山）
         ┊    │     │    │              │
  20%    ┊    │     │    ├── 组装制造   │
         ┊    │     │    │    │         │
   0%    └────┴─────┴────┴────┴─────────┴──►
            上游          中游        下游
```

**高壁垒+高毛利环节（重点关注）：**
- **半导体设备（光刻/刻蚀/薄膜）**：毛利率45-55%，国产化率低
- **EDA/IP**：毛利率>90%，但体量小
- **GPU/AI芯片设计**：毛利率60-70%，海光/寒武纪
- **HBM/存储接口**：毛利率40-50%，澜起科技等
- **核心材料（光刻胶/特种气体）**：毛利率40-60%

**中低壁垒环节（谨慎）：**
- **服务器组装**：毛利率10-15%，竞争激烈
- **封测代工**：毛利率15-25%，同质化
- **普通PCB**：毛利率20-30%，产能过剩

## 三、行业壁垒深度分析框架

### 七大真实壁垒（不是所有"壁垒"都是真的）

| 类型 | 真假鉴别 | 代表行业 |
|------|---------|---------|
| ✅ **技术专利壁垒** | 真——要看专利的不可绕过性 | 半导体设备、EDA |
| ✅ **认证周期壁垒** | 真——客户认证需要2-5年 | 军工、航空、汽车电子 |
| ✅ **规模效应壁垒** | 真——越做成本越低 | 晶圆代工、面板 |
| ✅ **网络效应壁垒** | 真——用户越多越好用 | AI平台、操作系统 |
| ⚠️ **资金壁垒** | 半真——有钱就能砸开 | 芯片制造（国家大基金介入后） |
| ❌ **先发优势壁垒** | 假——科技行业常被后来者颠覆 | 多数科技赛道 |
| ❌ **品牌壁垒(ToB)** | 假——企业客户只看性价比 | 工业品、元器件 |

### 壁垒的生命周期

```
壁垒强度
  ▲
  │  🔒 最强     ⬇ 侵蚀中     ⬇⬇ 快速消失
  │  专利期内的   专利到期后    技术路线被颠覆
  │  核心技术     竞争对手追上   新工艺取代旧工艺
  │
  └──────────────────────────────► 时间
```

**跟踪重点：** 哪些环节的壁垒在加强，哪些在减弱？加强的可以买入并持有，减弱的要考虑退出。
