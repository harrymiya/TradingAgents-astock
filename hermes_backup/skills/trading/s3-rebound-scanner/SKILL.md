# S3超跌反弹策略

## 核心理念

S3 = 深跌区 + 放量中阳线 + MA20深度偏离

超跌反弹的本质：**一只股票连续下跌跌到没人要了(MA20偏离-8%以下)，突然有一天放量拉出一根中阳线(涨3-7%)，说明有资金进场抄底。** 跟进去，持有5天，等反弹。

## 策略三要素匹配

| 要素 | 值 | 说明 |
|:----|:--|:-----|
| **选股频率** | 低频→中频 | 每日信号3-15只（不同市场状态不同） |
| **持有周期** | T+5 | 持有约5个交易日，让反弹充分展开 |
| **买卖节奏** | 尾盘确认→次日开盘买入 | 不追收盘价，不吃当日涨幅 |
| **仓位** | 每只15%，最多5只 | 分散风险，不平仓 |

## 优化参数（回测验证）

```
条件:
  20日位置(pos_20d) < 20     ← 处于近20日低位
  当日涨跌(chg): 3% ~ 7%     ← 放量中阳线（不是涨停）
  量比(vr_5): 1.2 ~ 2.5      ← 温和放量（不是天量）
  MA20偏离(ma20_pct): < -8%  ← ★核心参数:深跌远离MA20
  实体比例: < 70%             ← 不是大阳线追高（加分项）
```

**回测结果（3年全市场数据）：**

| 版本 | 信号数 | T+5收益 | 胜率 | 频率/日 |
|:----|:-----:|:-------:|:---:|:------:|
| 原版S3 | 9,237 | +3.13% | 60.6% | 12.7 |
| **优化版** | **4,036** | **+5.22%** | **71.1%** | **5.5** |
| 精选版 | 800 | +7.42% | 79.1% | 1.1 |

## 实战操作流程

### 每日流程

```
15:00  收盘
15:05  mootdx同步日K线到SQLite
16:00  收盘数据同步 + 多维度扫描（已有cronjob）
16:30  ★ S3扫描（当前cronjob）
        1. build_feat_table.py 重建特征表
        2. s3_scanner.py 扫描信号
        3. 结果推送飞书
16:45  看信号 → 决定明日操作
  T+1 09:25  开盘买入TOP 3-5只
  T+5 14:50  卖出
```

### 买入规则

- **买入时机**: 收盘确认信号后，次日开盘集合竞价买入
- **选股排序**: 评分系统
  - 实体比例<60% → +2分（温和放量）
  - 量比<2.0 → +1分（不过分放量）
  - 位置<15 → +1分（够低）
  - MA20偏离<-8% → +1分（深跌）
  - 涨<6% → +1分（不是涨停）
  - 连跌3天以上 → +1分（超跌企稳）
- **买入数量**: TOP 3-5只（信号少时3只，多时5只）

### 卖出规则

- **持有期**: T+5天（约5个交易日）
- **止损**: 单只-7%止损（盘中触发就卖）
- **不止盈**: 让利润奔跑，5天后自然卖出
