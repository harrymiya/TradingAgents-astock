## 一、核心心法总纲

### 炒股养家心法（情绪周期派）
- **信念第一**：全身心投入，渐悟到顿悟
- **赚钱效应/恐慌效应**决定大势：
  - 有赚钱效应 → 做热点为主
  - 有恐慌效应 → 做超跌为主
- **行情好就多做，行情不好就少做，要做就做最强，选股选大众情人**
- **买入机会，卖出风险**，不用止损止盈束缚自己
- 不同阶段不同手法：
  - **强势阶段** → 做最强个股主升浪，可多拿几天
  - **弱势阶段** → 超跌反弹，大部分第二天冲高就走
  - **平衡格局** → 持有热点板块观察变盘方向，试错加仓
- **操作本质**：通过试错单感受市场情绪，判断收益和风险，再决定是否奋力一击

### 92科比情绪周期
- 情绪周期四阶段：**启动→发酵→高潮→衰退**
- 不同周期的操作策略完全不同

### 退学炒股（心性修炼）
- **"我和小明"**：小明=内心的恐惧/贪婪/冲动，克服小明才能稳定盈利
- 交易是认识自我的过程

### 涅槃重升
- **控制回撤比追求盈利更重要**
- 稳定复利是唯一的道路

### 北京炒家（首板模式）
- 专注于**首板**，不打接力
- 严格分仓，单票不超过1/3
- **模式的一致性**比选到牛股更重要

## 二、情绪流龙头战法体系（杨楠）

### 三大规律
1. **情绪传递** → 追涨杀跌是群体本能，利用情绪周期
2. **强者恒强** → 只做强势股，远离杂毛
3. **物极必反** → 涨久必跌、跌久必涨，把握冰点转暖

### 人和篇（选股框架）
- **题材是股市第一生产力**
- 龙头股的**六大特征**：
  1. 最能带动板块
  2. 涨幅最大、最先涨停
  3. 换手充分
  4. 有题材支撑
  5. 盘子适中
  6. 人气最旺
- **龙头战法核心**：只做最强，只做龙头，不做跟风

### 地利篇（技术分析）
- **个股筹码结构**分析
- **多空能量转换**
- **MACD在情绪流中的运用**
- **主力踪迹发现**（少数派知识）
- 股票反转的常见走势

### 天时篇（择时）
- **择时重于择股，周期重于题材**
- 情绪周期的四个阶段识别
- 冰点→转暖→高潮→分化→退潮

### 情绪流龙头战法六大步骤
1. 大盘环境评估（安全/风险）
2. 热点板块识别
3. 板块内龙头确认
4. 买卖点判断
5. 仓位管理
6. 风控纪律

## 三、短线训练营实战技法

### 集合竞价看盘核心
- **9:20前不挂单**（可撤单，容易受骗）
- 竞价"量增价涨型" → 可参与
  - 竞价量≥昨日分时最大成交量一半
  - 不出现超3%的大跳档
- **弱转强竞价**：昨天烂板，今早竞价走强 → 连板概率大
- **反包竞价**：昨天涨停被砸/冲高回落，第二天竞价量增价高 → 低吸博弈反包

### 打板技术
- 打板的本质：**确认强势+次日溢价**
- **为什么要打板**：确定性最高，避免冲高回落
- **打板要点**：
  - 打板只打龙头，不打跟风
  - 关注涨停时间（越早越好）
  - 关注封单量
  - 关注换手率

### 低吸技术
- **强势股低吸**：龙头股分时回调不破关键均线
- **横盘龙头低吸**：龙头横盘缩量后的低吸
- **超跌低吸**：连续下跌后恐慌盘释放

### 妖股反包
- 识别龙头妖股特征
- 首次分歧后的反包机会
- 反包量能要求

## 四、闻少体系（情绪周期+盘口语言）

### 盘口语言终极解密

### 混沌龙头+起爆战法

- 量价关系在起爆点的应用

## 五、核心战法汇总

### 1. 首板战法（北京炒家）
- 专注首板，分仓操作
- 模式一致性是生命线
- 选股标准：有题材、有量、涨停时间早

### 2. 龙头接力（赵老哥）
- "八年一万倍"：专注龙头股主升浪接力
- 热点题材的龙头股 → 分歧转一致时介入
- 万亿大单扫货封板

### 3. 情绪周期低吸（炒股养家）
- 赚钱效应好时积极做多
- 恐慌效应时只做超跌
- 平衡市试错+加仓

### 4. 弱转强模式（闻少）
- 昨天烂板/分歧大，今天竞价强势 → 弱转强
- 昨天分歧，今天竞价放量高开 → 介入

### 5. 反包模式
- 龙头首阴后次日反包概率大
- 强势股涨停被砸 → 次日竞价量增价高 → 低吸反包

### 6. 横盘低吸（爱在冰川）
- 强势股回调横盘缩量 → 低吸潜伏
- 等待第二波拉升

### 7. 空间势能模式（综合）
- 价格回踩关键支撑（均线/中枢上沿）
- 缩量企稳+放量启动 → 介入

## 六、选股量化思路（代码实现方向）

### 情绪判断指标
1. 涨停板数量（今日/昨日对比）
2. 连板股数量（高位票情绪）
3. 跌停板数量（恐慌程度）
4. 昨日涨停股今日平均涨幅（赚钱效应）

### 龙头/强势股识别
1. 涨停时间（越早越强）
2. 封单金额/流通市值比
3. 换手率在合理范围（分歧换手板优于一字板）
4. 是否带动板块跟风

### 低吸/反包信号
1. 昨日涨停被砸/冲高回落 → 今日竞价放量高开 → 反包信号
2. 强势股连续3-5日回调 → 缩量企稳 → 低吸信号
3. 连板龙头首次分歧（断板但跌幅不大）→ 反包预期

### 仓位管理
1. 大盘趋势判断（多/空/震荡）
2. 情绪周期阶段（冰点/启动/高潮/退潮）
3. 根据以上两项确定仓位上限
